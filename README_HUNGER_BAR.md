## How to Use

1. **Open the Application**
   - Open `hunger_bar_cafe.html` in any modern web browser
   - Or serve it using a local server (recommended)

2. **Browse the Menu**
   - Scroll through three categories: Coffee, Food, and Desserts
   - Each item shows name, description, price, and image

3. **Add to Cart**
   - Click "Add to Cart" button on any item
   - View cart by clicking the floating cart icon (üõí) in bottom-right corner
   - Cart badge shows total number of items

4. **Manage Cart**

5. **Checkout**
